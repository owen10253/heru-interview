## Getting Started

First, run the backend server:

```bash
# to start backend service
cd backend

# then 
npm run start
# or
yarn dev
```

Then, run the development server:

```bash
# to start frontend service
cd frontend

# then 
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
